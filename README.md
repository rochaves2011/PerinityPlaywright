# Perinity Playwright Challenge

Este projeto implementa testes automatizados de login, produtos e carrinho utilizando Playwright com JavaScript e Page Object Model (POM).

## Requisitos

- Node.js
- npm

## Instalação

```bash
npm install
```

## Instalar o Playwright Browser

```bash
npx playwright install
```

## Executar os testes

```bash
npx playwright test
```

## Executar os testes em modo visível

```bash
npx playwright test --headed
```

## Estrutura do projeto

- pages/: classes com os elementos e ações das páginas
- tests/: cenários automatizados
- playwright.config.js: configuração do Playwright
- package.json: dependências e scripts
